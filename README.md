# Mini Pomodoro

An ultra-minimalist, zero-distraction Pomodoro timer designed for focus.

### Core Features
- **Fixed Cycles:** 25-minute work sessions, 5-minute optional breaks.
- **Daily Counter:** Tracks completed sessions with an automatic reset at 3:00 AM.
- **Zen Aesthetic:** Dark mode, Oura-inspired design with premium typography (DM Sans).
- **Meditative Alert:** A soft, synthesized chime plays on completion (no external audio files needed).
- **Ultra-Compact:** Responsive design that works perfectly in small floating windows (down to 200x200).
- **Zero Dependencies:** Pure vanilla HTML, CSS, and JS. No build step required.

### How to Use
Simply open `index.html` in any modern web browser. 

### Persistence
The session counter is saved to your browser's `localStorage` and persists across refreshes.
